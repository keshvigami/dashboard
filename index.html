Alpine.js Example

x-data

Initialize Alpine state on an element.

Count:
<div x-data="{ count: 0 }">
  <span x-text="count"></span>
  <button @click="count--">Decrement</button>
  <button @click="count++">Increment</button>
</div>

x-text

Dynamically set text content.

<span x-text="title"></span>

x-on

Attach event listeners to elements.

<button @click="toggleOpen">Toggle Section</button>

x-show, x-cloak, x-transition

Show/hide with transitions and prevent FOUC.

This section is visible when isOpen is true.
<div x-show="isOpen" x-cloak x-transition>...</div>

x-if

Conditionally render elements in the DOM.

<template x-if="isOpen">...</template>

x-bind

Dynamically bind attributes.

<button :class="isOpen ? 'bg-red-500' : 'bg-green-500'">...</button>

x-html

Render raw HTML content.

<div x-html="htmlContent"></div>

x-for

Loop over arrays or objects.

<template x-for="item in items" :key="item"><li x-text="item"></li></template>

x-model

Two-way data binding for form elements.

Input:

<input x-model="inputValue" ... >

x-effect

Run code when dependencies change.

Check console for x-effect

<p x-effect="console.log('Input changed to:', inputValue)">...</p>

x-ignore

Ignore a subtree from Alpine reactivity.

This block is ignored by Alpineâ€™s reactivity.

<div x-ignore>...</div>

x-ref + $refs

Reference elements and access them in JS.

<input x-ref="myInput" ... >
<button @click="$refs.myInput.focus()">...</button>

x-teleport

Teleport content to another part of the DOM.

<template x-teleport="body">...</template>

x-id

Generate unique IDs for accessibility.

<div x-id="['input']">...</div>

Alpine Store

Global reactive state with Alpine.store.

Counter:

Alpine.store('counter', { count: 0, increment() { this.count++ } })

Async Example

Fetch data asynchronously and update state.

async fetchData() { ... }

$watch

Watch data changes and react.

Watching inputValue: check console!

this.$watch('inputValue', value => ...)

$dispatch

Dispatch custom events.

Event received! Open = true
$dispatch('custom-event', true)

$el, $data, $root

Access element, data, and root references.

console.log($el, $data, $root)

Nested x-data

You can nest Alpine components for isolated state.

Outer:
Inner:
<div x-data="{ outer: 1 }">
  ...
  <div x-data="{ inner: 10 }">...</div>
</div>

Computed Property

Use a getter for computed values.

+ =
<div x-data="{ a: 2, b: 3, get sum() { return this.a + this.b } }">...</div>

x-init (Async)

Run async code on component init.

x-init="getJoke()" (async function)

x-transition (Custom)

Custom transition classes for animation.

Custom fade & scale transition!
x-transition:enter/leave with custom classes

x-for (Objects)

Loop over an array of objects.

x-for="user in users" :key="user.id"

x-model (Checkbox, Radio, Select)

Bind to checkboxes, radios, and selects.

x-model with checkbox, radio, select
